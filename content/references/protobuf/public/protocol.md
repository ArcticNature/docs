Node information
----------------

  * The user sends a
    [`NodeInfoRequest`]({{< relref "references/protobuf/public/messages.md#nodeinforequest" >}})
    message to the Manager.
  * The Manager sends a
    [`NodeInfoResponse`]({{< relref "references/protobuf/public/messages.md#nodeinforesponse" >}})
    message back.
